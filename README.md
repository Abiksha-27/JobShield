# JobShield

JobShield is a full-stack scam detection and company trust verification platform developed for freshers and job seekers.

## Features
- Company Trust Verification
- Scam Reporting System
- Dynamic Trust Score Calculation
- User Reviews
- Login & Registration
- Responsive Fintech UI

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- REST APIs

## Purpose
To help users identify fake job offers, fake internships, and suspicious companies.

## Future Improvements
- AI Scam Detection
- OTP Verification
- Screenshot Proof Upload
- Admin Moderation
